# 修士論文 研究計画：予測考慮型計画の実データ較正・検証（軸A）

作成 2026-06-15。AVEC'26 論文（`docs/FINAL_BENCHMARK_REPORT.md`、提出済み）の発展。
**改訂 2026-06-15（rev2）**：code-review（xhigh）の10指摘を反映 — 循環性・速度域ミスマッチ・分布シフト交絡への対処、検証設計の再構成、限界節の新設、事実誤り（ETH/UCY 生データの所在）とライセンス確認の追加。

## 1. 背景と位置づけ

AVEC 論文の核心的主張は3つだが、**すべて SFM シミュレータ内**で得られている：

1. 変位誤差（ADE/FDE）は分布を持たない CV を最良と順位づける — が、これは**シミュレータへの適合の人工物**で実予測品質ではない。
2. ランダム化 GT 下で**衝突するのは CV のみ**（学習系は衝突ゼロ）＝「分布フリー予測の危険」。
3. 全サンプルを使う **robust 機会制約計画**は全シナリオで margin を稼ぎ、一様 inflation では再現できない。

論文最大の弱点は **(L1) SFM 斥力パラメータがシナリオ固有・恣意的で実データ未検証**であること。修論はこれを実データで突く。

**本研究で「実データ検証」が意味する範囲（誇張しないための明確化）**：
- **主たる実データ証拠 = 開ループ予測精度**（RQ1a）。実歩行者軌道に対し予測器を直接評価する、交絡の少ない検証。
- **較正（RQ2）が接地するのは SFM の自車斥力*パラメータ*（σ, v0）のみ**。SFM の*関数形*、シナリオ幾何、歩行者目標、歩行者間力は合成のまま。よって本研究は「2スカラーの実データ較正＋その下での頑健性検証」であり、シナリオ全体の実データ再現ではない（§9 に限界を明記）。
- **閉ループ再ベンチ（RQ1b）は独立した外的検証ではなく、較正済み反応モデル下での*感度・頑健性チェック*** と位置づける。較正した GT に対して計画器を評価する以上、AVEC sim 実験と機構を共有する（循環性。§7・§9 参照）。独立性は RQ1a の開ループ予測証拠が担う。

この正直な再フレーミングにより、「robust 利得は実データ較正後も残るか」を*頑健性の主張*として、「序列はシミュレータ人工物か」を*開ループの実データ主張*として、役割を分離する。

## 2. リサーチクエスチョンと仮説

- **RQ1（外的妥当性・主証拠）**：実歩行者分布で予測器の序列はどうなるか。
  - **RQ1a-ETH/UCY（主軸）**：ETH/UCY は SGAN の学習元で leave-one-out 評価が成立する（in-domain・クリーン）。ここで CV<LSTM<SGAN の序列が崩れる/縮小するかを測る。**これが交絡の最も少ない実データ検証**。
  - RQ1a-DUT（副・交絡注意）：DUT 適用は座標/シーン/スケールのシフト＋未学習の車両影響を伴うため、結果は RQ3（適応）とセットで「人工物か領域外か」を分離して解釈する。単独では H1 の診断に使わない。
  - 仮説 H1：in-domain（ETH/UCY）でも CV の ADE 優位が縮小すれば「序列＝SFM 過適合」を支持。
- **RQ2（自車斥力の較正）**：SFM 自車斥力は車両接近への実回避を再現するか。(σ, v0)（+速度依存）を**CITR（制御実験）主体**で同定し、DUT で汎化検証できるか。
  - 仮説 H2：論文の手調整値（既定 σ, v0 = 0.7/3.5、`integrated_simulator.py:75-76`）は実測と乖離する。較正で回避開始距離・最小車間距離分布が実データに整合する。**ただし速度依存を入れない限り運用速度域（5-6 m/s）への外挿は無効**（下記対処）。
- **RQ1b（頑健性チェック・本命の応用）**：較正済み反応モデル下で robust/inflation/single を再ベンチし、margin 利得・CV 危険性が*残るか*。外的検証ではなく感度分析。
- **RQ3（ストレッチ・かつ RQ1a-DUT の交絡分離に必須）**：予測器を DUT へ適応し、序列の変化と「領域外 vs 人工物」の切り分け。

いずれの結果も論文化可能（H1 が崩れる＝知見の転移限界という負の結果も価値あり）。

## 4. 方法論（既存コードの再利用と新規実装を明記）

### 新規実装（W1-2 に計上、現状リポジトリに不在）
- `ReplayPedestrianSource`：`PedestrianSimulator`（`src/simulation/integrated_simulator.py`）と差し替え可能な、記録軌道を正解として供給するクラス。
- 座標/単位変換：DUT/CITR の px→m（提供 ratio ファイル）、ETH/UCY world 座標 → SGAN 入力規約（8 obs @0.4s）。既知物理量（歩行速度 ~1.3 m/s・車幅）で sanity check。
- 系同定オプティマイザ（RQ2 用、現状なし）。
- **忠実度指標群**：回避開始距離・最小車間距離分布・KS 検定・ロールアウト ADE。`src/core/metrics.py` は ADE/FDE/minADE/KDE-NLL のみ実装のため**新規追加が必要**。

### RQ1a — 開ループ domain-gap 検証
- `ReplayPedestrianSource` → 既存 `PedestrianObserver`（0.4s ダウンサンプリング）→ `TrajectoryPredictor`（cv/lstm/sgan）→ `metrics.py`（ADE/FDE/minADE/NLL）を再利用。**ego を介さない純予測評価**なので、リプレイ時の自車反応不整合は生じない。
- 主軸は ETH/UCY（in-domain）。DUT は副（RQ3 とセット解釈）。出力を sim 結果（`output/benchmark_rand_*_g5a2`）と対比。

### RQ2 — 自車斥力の較正（系同定）
- **較正対象**：`ego_repulsion_sigma`/`ego_repulsion_v0`（既定 0.7/3.5、`integrated_simulator.py:75-76`、`social_force_params` 経由で上書き可）。
- **同定（CITR 主体）**：(歩行者 位置・速度, 車両 位置・向き・速度) を抽出。CITR の制御 VCI は歩行者間/目標力が小さい構図を選び、残差を自車斥力に帰属させやすくする。手法は (A) 1ステップ加速度マッチング、(B) 短時間ロールアウト ADE 最小化。
- **交絡対処（#3/#5）**：DUT は目標が未観測で残差に歩行者間/目標力が混入するため**同定には使わず汎化検証のみ**。CITR で同定→DUT で held-out 検証。可能なら歩行者間力は pysocialforce 既定で固定し自車項のみ自由化。
- **速度域対処（#2 必須）**：CITR/DUT 車両は低速（~1-3 m/s）、AVEC 自車は 5-6 m/s（scenario 6.0/5.0/5.0）。次のいずれかを採用し明記する：(i) 斥力に相対速度依存項を追加して同定、(ii) 較正を計画器運用速度に近い接近サンプルへ層別、(iii) 低速域での較正と明記し RQ1b の運用速度を CITR/DUT 域に制限。**速度外挿の妥当性を論証せずに 6 m/s へ適用しない**。
- **対照群（#6 必須）**：(a) ホールドアウト負対数尤度、(b) 帰無/ランダム摂動パラメータ対照、(c) 非SFM ベースライン（等速反応なし等）と比較。「実回避物理を捉えた」か「2スカラーの最小二乗 fit」かを区別する。残差データ駆動補正は小規模データ（CITR 340/DUT 1793）ゆえ過学習注意・held-out で検証。

### RQ1b — 較正済みモデル下での頑健性チェック
- 較正値を `social_force_params` に投入し、AVEC キャンペーンを再実行：`run_statistical_benchmark.py`（rand GT）＋ `run_da_poc.py`（robust/single/inflation）＋ margin_control。
- **位置づけは感度分析（#1）**：較正した SFM-GT に対する評価＝AVEC sim と機構共有。独立検証ではなく「結論が較正下でも保持されるか」を測る。独立性は RQ1a の開ループ証拠が担保。
- 評価：MinDist・衝突（envelope breach）・走行時間・乗り心地・TTC、robust/single/inflation の Δ、Welch/Fisher（AVEC と同手順）。sim 既定版と並置。

### RQ3 — 予測器のドメイン適応（ストレッチ／交絡分離）
- ETH/UCY 学習 SGAN を DUT へ fine-tune（zara→DUT 転移）。best-of-N ADE 序列の変化を測り、RQ1a-DUT の「領域外 vs 人工物」を分離。

## 5. 評価指標

- **予測**：ADE/FDE、minADE/minFDE（best-of-20）、KDE-NLL（`metrics.py` 既存）。
- **較正忠実度（新規実装）**：ロールアウト ADE/FDE、回避開始距離、最小車間距離分布の一致（KS）、held-out NLL＋対照群。
- **計画（閉ループ）**：MinDist、衝突率、走行時間、RMSJerk/MeanAcc、MinTTC、robust/single/inflation の Δ。
  - **指標の注意（#8）**：「衝突」は R_ego+R_ped=1.2 m（ego 1.0+ped 0.2、ハードコード）の包絡侵食＝点車両の意図的過大近似。実車両形状ではないため**絶対的な実車安全値ではなく、条件間の相対比較**として解釈・記述する。
- **ドメインギャップ定量**：既定 σ,v0 vs 較正 σ,v0、予測器序列の sim vs real。

## 7. リスクと意思決定点

- **循環性（#1）**：RQ1b は較正 GT に対する自己整合評価。→ 主証拠を RQ1a 開ループに置き、RQ1b は感度分析と明記。判断不要（設計で対処済）。
- **速度域外挿（#2）**：低速較正→高速運用。→ 速度依存項 or 層別 or 運用速度制限のいずれかを W7-12 で確定。
- **同定可能性（#3/#5）**：DUT の力分解が曖昧。→ CITR 主体・DUT 検証限定。W12 で CITR fit が DUT で破綻するなら速度/未モデル化力の切り分けを追加。
- **分布シフト交絡（#4）**：DUT への SGAN 適用。→ RQ1a 主軸を ETH/UCY に、DUT は RQ3 とセット。
- **較正の見かけ改善（#6）**：→ 対照群（held-out NLL・帰無パラメータ）必須。
- **工数（#7）**：新規実装（Replay・変換・系同定・忠実度指標）を W1-3 に明示計上。座標バグは物理量で sanity check。
- **負の結果**：知見が転移しないこと自体を主結果に。

3. **実データによる予測器序列の検証**（RQ1a、主証拠）。
4. **双方向反応モデルの実データ較正**（RQ2、方法論的貢献）。
5. **較正済みモデル下での予測考慮型計画の頑健性評価**（RQ1b、感度分析）。

貢献（誇張を避けた表現）：(i) ped-vehicle 相互作用 SFM 自車斥力の実データ較正手法（対照群つき）、(ii) **開ループ実データ上での**変位誤差序列のシミュレータ人工物性の確認、(iii) robust 機会制約計画の利得が較正済みモデル下でも保持される（or されない）ことの感度分析。後続の国際会議論文の素地。

## 9. 限界（修論本文に明記する／rev2 で新設・拡充）

- **部分的較正**：較正するのは自車斥力 (σ, v0) のみ。SFM 関数形・歩行者間力・歩行者目標・シナリオ幾何は合成のまま。「実データ較正」はこの範囲を指す（全シナリオの実データ再現ではない）。
- **閉ループの非独立性**：RQ1b は較正 GT に対する評価＝外的検証ではなく感度分析。独立な実データ閉ループ安全検証は本研究の範囲外。
- **速度域**：較正は低速 DUT/CITR 由来。運用速度への適用は採用した速度対処の前提に条件付き。
- **包絡指標**：1.2 m 衝突円盤は合成の過大近似。実車形状（multi-circle、AVEC で将来課題化）は扱わない。
- **データ規模/単一性**：CITR/DUT は小規模・特定環境。クロス環境頑健性は今後の課題。
