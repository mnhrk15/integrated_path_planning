## 3. 教訓・注意

- macOS BSD grep はパターン中の `^` をアンカー扱いすることがあり、`grep -c 'd_t^s'` が**偽陰性**を返した（JA の 1 箇所が一時未適用のまま検証をすり抜けかけた）。文字列照合は `grep -F` を使うこと。
- EN のページ予算は**スラックほぼゼロ**（1行の追加で7ページ化）。今後の文追加は microtype 下でも要リビルド確認。
- `welch_vs_baseline.csv` の `delta_vs_base` は**全条件とも SGAN single (inf1.00) 基準**。LSTM robust−LSTM single は差分の差で求める（例: S1 1.3735−0.0702=+1.30）。
